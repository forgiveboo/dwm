#### 基于dwm6.2打过补丁的dwm
#### 使用的补丁
`dwm-alpha-20201019-61bb8b2.diff` 透明补丁
`dwm-autostart-20210120-cb3f58a.diff` 自动启动脚本
`dwm-awesomebar-6.2.diff` 状态栏展示多个任务
`dwm-hide-and-restore.diff` 隐藏和重新展示任务
`dwm-hide_vacant_tags-6.2.diff` 隐藏没有任务的tag
`dwm-noborder-6.2.diff` 当只有一个窗口时去掉边框
`dwm-rotatestack-20161021-ab9571b.diff` 改变栈的顺序
`dwm-scratchpad-6.2.diff` 拥有一个随时打开的浮动终端
`dwm-vanitygaps-20200610-f09418b.diff` 提供窗口之间的边框
`dwm-viewontag-20210312-61bb8b2.diff` 使用户跟随到tag
